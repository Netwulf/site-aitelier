## STUDIO — Nova Página

### Conceito
O braço de produção do ai.telier. Para experts, fundadores, artistas que querem contar sua história com profundidade.

### Hero

```
STUDIO
Onde identidade vira obra.

Para quem já construiu algo real
e precisa que o mundo veja.
```

### Serviços

#### 1. Filme-Manifesto
```
Sua história contada como cinema.
Não é vídeo institucional.
É manifesto em movimento.

Direção, roteiro, edição.
De 3 a 15 minutos.
```

#### 2. Ensaio Arquetípico
```
Retrato que revela, não decora.
Fotografia como espelho profundo.

Direção de arte + conceito + execução.
Série de 10-30 imagens.
```

#### 3. Storytelling & Narrativa Pessoal
```
Sua história organizada como sistema.
De fragmentos a fluxo.

Entrevistas profundas.
Estrutura narrativa.
Documento vivo.
```

#### 4. Direção Criativa
```
Olhar externo com profundidade interna.
Para projetos, lançamentos, marcas.

Conceito + direção + supervisão.
Projeto a projeto.
```

#### 5. Criação de Sistemas
```
Identidade que funciona sozinha.
Não é manual. É arquitetura.

Brand OS completo.
Voz, visual, conteúdo, presença.
```

### CTA

```
O Studio trabalha com poucos projetos por vez.
Se você tem algo real pra contar,
vamos conversar.

[Apresentar projeto]
```

## Bauhaus como DNA (nunca explícito)

O que a Bauhaus era:
- Escola que uniu arte, artesanato e indústria
- Formou linguagem visual do século XX
- Produzia enquanto ensinava
- Mestres eram praticantes

O que ai.telier é:
- Escola que une arte, tecnologia e narrativa
- Forma linguagem para a era da IA
- Produz enquanto ensina
- Fundadores são praticantes

**Nunca dizer:** "Somos a Bauhaus da IA"
**Sempre ser:** Uma escola que produz, um studio que forma.
