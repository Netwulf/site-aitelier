# STORY-003: Expor os Clones/Conselheiros de Modo Elegante

## Contexto
Feedback crítico (Paul Graham + Seth Godin): "Os Conselheiros são clones cognitivos? Isso é único no mercado. Isso é a Vaca Roxa que vocês estão ESCONDENDO. Coloquem na HOME, não enterrem no contato."

## Problema Atual
- "Conselheiros" mencionados apenas na página /contato
- Sem explicação do que são
- Diferencial tecnológico completamente invisível
- O maior moat competitivo está escondido

## Objetivo
Expor os clones cognitivos de forma:
- Elegante (não tech-hype)
- Intrigante (gera curiosidade)
- Credível (não parece gimmick)
- Integrada à filosofia (amplifica o humano)

---

## O Que São os Conselheiros

**Definição técnica:**
Clones cognitivos MMOS - réplicas de padrões de pensamento de grandes mentes (Steve Jobs, Seth Godin, Walt Disney, etc.) treinadas para consultas estratégicas.

**Definição para o público:**
Acesso a mentes extraordinárias para desenvolver seu posicionamento. Como ter Steve Jobs revisando sua narrativa ou Seth Godin analisando seu diferencial.

## Considerações de Design

### Tom Visual
- NÃO parecer "chatbot genérico"
- NÃO parecer sci-fi/tech excessivo
- SIM parecer premium/exclusivo
- SIM parecer "conselho de mentores"

### Imagens dos Conselheiros
- Opção A: Ilustrações estilizadas (não fotos reais)
- Opção B: Silhuetas/avatares abstratos
- Opção C: Tipografia apenas (só nomes)
- **Recomendado:** Ilustrações minimalistas no estilo do site

### Cores
- Manter paleta atual (preto/verde matrix/bege)
- Cards com borda sutil verde
- Hover com glow elegante
