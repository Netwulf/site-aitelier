# Story 3.2: Cards & Hover Polish

**Story ID:** AITELIER-3.2
**Epic:** 3.0 - UX Polish Framer/Apple
**Priority:** P0
**Estimate:** 4-5 hours
**Status:** Ready for Development
**Depends on:** Story 3.1

---

## User Story

**As a** visitor interacting with cards and buttons
**I want** satisfying hover feedback that feels premium
**So that** every interaction reinforces the high-quality brand perception

---

## Acceptance Criteria

### AC1: Card Lift Effect
- [ ] Cards lift on hover (translateY: -8px)
- [ ] Shadow expands on hover (deeper, larger spread)
- [ ] Transition: 300ms cubic-bezier(0.4, 0, 0.2, 1)
- [ ] Smooth return to original state

### AC2: Triptych Scale & Glow
- [ ] Slight scale on hover (1.02)
- [ ] Subtle inner glow appears
- [ ] Number indicator brightens (opacity 0.2 → 0.5)
- [ ] Bottom accent line animation maintained

### AC3: Button Shine Effect
- [ ] `.btn-ghost` has sweep shine on hover
- [ ] Shine moves from left to right
- [ ] Duration: 400ms
- [ ] Only visible during hover state

### AC4: Glass Card Enhancement
- [ ] `.glass` cards have subtle border glow on hover
- [ ] Background opacity increases slightly
- [ ] Transform: translateY(-2px) maintained
- [ ] Inner shadow added for depth

### AC5: Focus Ring Animation
- [ ] All interactive elements have visible focus ring
- [ ] Ring animates in (scale from 0.95 → 1)
- [ ] Color: tech-olive with opacity
- [ ] Offset: 2px from element
- [ ] Works with keyboard navigation

## Technical Implementation

### Task 1: Card Lift Effect

**File:** `src/index.css`

```css
/* Add to @layer components */
.card-lift {
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1),
              box-shadow 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.card-lift:hover {
  transform: translateY(-8px);
  box-shadow:
    0 20px 40px rgba(0, 0, 0, 0.3),
    0 10px 20px rgba(0, 0, 0, 0.2);
}

/* For reduced motion */
@media (prefers-reduced-motion: reduce) {
  .card-lift:hover {
    transform: none;
  }
}
```

### Task 3: Button Shine Effect

**File:** `src/index.css`

```css
/* Update .btn-ghost */
.btn-ghost {
  @apply border-2 border-matrix-green bg-transparent text-matrix-green
         font-bold px-8 py-4 uppercase tracking-wider
         transition-all duration-300;
  position: relative;
  overflow: hidden;
  will-change: transform;
  transform: translateZ(0);
}

/* Background fill on hover */
.btn-ghost::before {
  content: '';
  @apply absolute inset-0 bg-matrix-green;
  transform: translateX(-100%);
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* Shine sweep effect */
.btn-ghost::after {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 50%;
  height: 100%;
  background: linear-gradient(
    90deg,
    transparent,
    rgba(255, 255, 255, 0.3),
    transparent
  );
  transition: none;
}

.btn-ghost:hover::before {
  transform: translateX(0);
}

.btn-ghost:hover::after {
  animation: shine-sweep 0.4s ease-out;
}

.btn-ghost:hover {
  @apply text-brutal-black;
}

.btn-ghost span {
  position: relative;
  z-index: 1;
}

@keyframes shine-sweep {
  0% { left: -100%; }
  100% { left: 150%; }
}

@media (prefers-reduced-motion: reduce) {
  .btn-ghost:hover::after {
    animation: none;
  }
}
```

### Task 5: Focus Ring Animation

**File:** `src/index.css`

```css
/* Add to @layer base or utilities */

/* Custom focus ring for all interactive elements */
:focus-visible {
  outline: none;
}

a:focus-visible,
button:focus-visible,
input:focus-visible,
select:focus-visible,
textarea:focus-visible,
[role="button"]:focus-visible {
  outline: none;
  box-shadow:
    0 0 0 2px hsl(var(--background)),
    0 0 0 4px hsl(var(--matrix-green) / 0.6);
  border-radius: 2px;
}

/* Animated focus ring */
@supports (animation: focus-ring-in 0.2s) {
  a:focus-visible,
  button:focus-visible,
  [role="button"]:focus-visible {
    animation: focus-ring-in 0.2s ease-out;
  }
}

@keyframes focus-ring-in {
  from {
    box-shadow:
      0 0 0 2px hsl(var(--background)),
      0 0 0 2px hsl(var(--matrix-green) / 0);
    transform: scale(0.98);
  }
  to {
    box-shadow:
      0 0 0 2px hsl(var(--background)),
      0 0 0 4px hsl(var(--matrix-green) / 0.6);
    transform: scale(1);
  }
}

@media (prefers-reduced-motion: reduce) {
  a:focus-visible,
  button:focus-visible,
  [role="button"]:focus-visible {
    animation: none;
  }
}
```
